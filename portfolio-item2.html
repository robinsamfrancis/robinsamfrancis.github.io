First Aid HelperGoogle Assistant Chatbot

Abstract

Performing first aid before an ambulance arrives can often be the difference between life and death. So this chatbot is conversational and interactive, where the decisions that the user makes can save someone's life.

This helps the user to find the correct first aid that must be taken in case of any emergency. It's action on google which can be invoked on any android phone having Google Assistant by saying "Talk to First Aid Helper" Link
